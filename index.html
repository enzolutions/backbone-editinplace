Backbone.EditInPlaceView

An edit-in-place widget built with Backbone.js

Example

HTML

<div id="edit-in-place"></div>

Model

var Status = Backbone.Model.extend();
var status = new Status({
    "text: "Don't look to the eyes of a stranger"
});

View

new EditInPlaceView({
    el: $("#edit-in-place"),
    model: status,
    attribute: "text"
}).render();
